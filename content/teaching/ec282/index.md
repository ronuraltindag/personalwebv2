<details>
<summary><strong>Software & Tools</strong></summary>

You'll use **R** as the primary language. Follow the steps below to get set up.

### Step 1: Install R (the language)

1. Go to [cloud.r-project.org](https://cloud.r-project.org/)
2. Click your operating system:
   - **Windows:** Click "Download R for Windows" → "base" → "Download R-4.x.x for Windows"
   - **Mac:** Click "Download R for macOS" → Choose the `.pkg` file for your Mac (Apple Silicon or Intel)
3. Run the installer and accept all defaults

### Step 2: Install RStudio (the editor)

1. Go to [posit.co/download/rstudio-desktop](https://posit.co/download/rstudio-desktop/)
2. Scroll down and click the download button for your operating system
3. Run the installer and accept all defaults
4. Open RStudio - it will automatically find R

### Step 3: Test your installation

Open RStudio and type this in the Console (bottom left):
```r
1 + 1
```
If you see `[1] 2`, you're all set!

### Alternative editors (optional)

- **VS Code** - Versatile editor with R extensions
- **Cursor** - AI-powered editor for writing and debugging
- **AI assistants** (Claude, ChatGPT) - For learning and troubleshooting

<summary><strong>Topics Covered</strong></summary>

1. **Introduction to Econometrics & R** - Getting started with data
2. **Simple Regression** (Ch. 2) - OLS, interpretation, functional forms
3. **Multiple Regression: Estimation** (Ch. 3) - Adding variables, mechanics of OLS
4. **Multiple Regression: Inference** (Ch. 4) - t-tests, F-tests, confidence intervals
5. **Qualitative Variables** (Ch. 7) - Dummy variables, interactions, LPM
